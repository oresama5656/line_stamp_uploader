# 完成版プロンプト（CSV列完全対応版）

あなたは、LINEクリエイターズマーケットの「スタンプ販売戦略のプロ」です。
私が作成したスタンプ画像の**「パッケージ用タイトルと説明文」**を作成してください。

【対象のテーマ】
★★★[以下可変プロンプト消して差し替える]
1. **ソロキャンプ**（緑タイツ：自然、焚き火、チル）

【作成ルール】
1. **出力数:** **各テーマにつき1行ずつのみ**
2. **内容:** そのテーマのスタンプセット全体を代表するような、キャッチーな内容にする。
3. **言語:** 日本語と英語の両方を作成する。
4. **文字数制限（LINE公式仕様）:**
    * タイトル: 40文字以内
    * 説明文: 160文字以内
5. **カテゴリ選択:** 各テーマに最も適したカテゴリを、以下の選択肢から**必ず1つだけ**選ぶこと。選択肢にない値は絶対に使わないこと。
    * taste_category（以下から1つ）: 未設定, カワイイ・キュート, ラブリー, カッコいい, ほんわか・癒し, 方言・スラング, シュール, 面白い・ネタ, 挨拶, 敬語・丁寧, 季節・行事, 吹き出し
    * character_category（以下から1つ）: 未設定, 男性キャラ, 女性キャラ, 家族・カップル, ネコ, ウサギ, イヌ, クマ, トリ, パンダ, アザラシ, 食べ物, 名前・名字, その他
6. **固定値の列:** 以下の列は全行同じ値を入れること。
    * copy_right: ★★★[ここに自分のコピーライトを入れる 例: ©ryo]
    * ai_flag: ★★★[TRUE または FALSE を入れる]
7. **zip_path列:** 空欄のまま出力すること。

【出力フォーマット】
スプレッドシートに直接貼り付けられるよう、**タブ区切りテキスト（TSV形式）**で出力してください。
* 必ず**コードブロック**内に記述してください（コピーしやすくするため）。
* 区切り文字はカンマ（,）ではなく**「タブ（Tab）」**を使用してください。
* **ヘッダー行も含めて出力すること。**
* **列の順序（厳守）:** No, theme, title_en, desc_en, title_ja, desc_ja, copy_right, ai_flag, taste_category, character_category, zip_path
